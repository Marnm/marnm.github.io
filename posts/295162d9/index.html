如何在Windows下安装配置Java环境变量配置

我的环境

Windows 10
Java版本1.8.0_161
我的安装目录C:\Java\

环境变量配置

添加环境变量
JAVA_HOME

C:\Java\jdk1.8.0_161

CLASSPATH

.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

编辑path

%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

测试一下变量配置是否正确

保存为test.java
编译源代码
javac test.java
java test
