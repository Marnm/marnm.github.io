Shell的基本运算符

发表于 2018-08-10
分类：
  • Linux
·
阅读本文可能花费您 6 分钟

Shell 和其它编程语言一样，支持多种运算符，包括：

  • 算数运算符
  • 关系运算符
  • 布尔运算符
  • 字符串运算符
  • 文件测试运算符
    原生bash不支持简单的数学运算，但是可以通过其它命令来实现，例如awk和expr最常用。
    expr是一款表达式计算工具，使用它能完成表达式的求值操作。

    算数运算符

    相当于小学知识
运算符 说明 举例(a=10,b=20),``不是单引号
+ 加 `expr $a + $b`结果为30
- 减 `expr $b - $a`结果为10
* 乘 `expr $a \* $b`结果为200
/ 除 `expr $b \ $a`结果为2
% 取余 `expr $a % $3`结果为1
= 赋值 `expr $a = $b`结果为false
== 等于 `expr $a == $b`结果为false
!= 不等于 `expr $a != $b`结果为true

关系运算符

关系运算符只支持数字，不支持字符串，除非字符串的值是数字。

运算符 说明 举例(a=10,b=20),``不是单引号
-eq 检测两个数是否相等，相等返回true [ $a -ea $b]返回false
-ne 检测两个数是否不相等,不相等返回true [ $a -ne $b ]返回true
-gt 检测左边的数是否大于右边的，是，则返回true [ $a -gt $b ]返回false
-lt 检测左边的数是否小于右边的，是，则返回true [ $a -lt $b ]返回true
-ge 检测左边的数是否大于等于右边的，是，则返回true [ $a -ge $b ]返回false
-le 检测左边的数是否小于等于右边的，是，则返回true [ $a -le $b ]返回true

布尔运算符

运算符 说明 举例(a=10,b=20),``不是单引号
! ‘非’运算，把条件测试中的判断结果取相反值 [ !false ]返回true
-o ‘或’运算，有一个表达式为true则返回true [ $a -lt 20 -o $b -gt 100]返回true
-a ‘与’运算，两个表达式都为true才返回true [ $a -lt -a $b -gt 100 ]返回false

字符串运算符

运算符 说明 举例(a=10,b=20),``不是单引号
= 检测两个字符串是否相等，相等返回true [$a = $b]返回false
!= 检测两个字符串是否相等，不相等返回true [$a != $b ]返回true
-z 检测字符串长度是否为0，为0返回true [ -z $a ]返回false
-n 检测字符串长度是否为0，不为0返回true [ -n “$a” ]返回true
str 检测字符串是否为空，不为空返回true [ $a ]返回true

文件测试运算符

运算符 说明 举例(a=10,b=20),``不是单引号
-b 测试文件是否为块设备文件 [ -b $file ]返回false
-c 测试文件是否为字符设备文件 [ -c $file ]返回false
-d 测试文件是否为目录类型 [ -d $file ]返回false
-f 测试文件是否为普通文件（既不是目录，也不是设备文件） [ -f $file]
-g 测试文件是否设置了SGID位
-k 测试文件是否设置了粘着位(Sticky Bit)
-p 测试文件是否有名管道
-u 测试文件是否设置了SUID位
-r 测试文件是否可读
-w 测试文件是否可写
-x 测试文件是否可执行
-s 测试文件是否为空
-e 测试文件（包括目录）是否存在
