Markdown

1. 粗体和斜体

使用 * 和 ** 表示粗体和斜体

示例：

这是 斜体 ，这是 粗体 。

2. 分级标题

使用 === 表示一级标题， 使用 — 表示二级标题。

示例：

你也可以选择在行首加#号表示不同级别的标题（H1-H6），例如，# H1,，## H2， ### H3，#### H4。

3. 超链接

使用[描述](链接地址)为文字增加超链接

示例：

这是去玩本人博客的连接

4. 无序列表

使用 *，+，- 表示无序列表。

示例

  • 无序列表项一
  • 无序列表项二
  • 无序列表项三

5. 有序列表

使用数字和点表示有序列表

示例：

  1. 有序列表项一
  2. 有序列表项二
  3. 有序列表项三

6. 文字引用

使用 > 表示文字引用

示例：

君不见，黄河之水天上来，奔流到海不复回。

天生我材必有用，千金散尽还复来。
