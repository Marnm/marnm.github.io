vmtools

如果在VMware虚拟机里面，可以用open-vm-tools，这是原来的vmtools通过开源出来的版本而已

sudo apt-get install open-vm-tools open-vm-tools-desktop

gnome-teawk-tool

安装gnome-tweak修改DPI、主题、图标

sudo apt-get install gnome-tweak-tool

sudo apt-get update

开发环境
JAVA

apt install openjdk-8-jdk
apt install openjdk-8-source

oracle-jdk
官网下载JDK
sudo tar -zxvf jdk-8-xxx.tar.gz -C /usr/
环境变量配置

cat >> ~/.bashrc <<- "EOF"
export JAVA_HOME=/usr/jdk-xxxx...
export PATH=$JAVA_HOME/bin/$PATH
EOF

source ~/.bashrc

Qv2ray核心文件配置
由于某些众所周知的政策原因，Qv2ray 本身并 不包含 V2Ray 核心可执行文件。这些核心文件来自一个叫做 v2ray-core 的项目，需要用户手动下载安装到指定位置。
项目地址https://github.com/v2fly/v2ray-core/releases
将解压出来的核心文件放到.config/qv2ray/vcore下
v2ray | v2ctl | geoip.dat | geosite.dat （这几个必须）

出现很多命令丢失的问题，命令无法使用
在/etc/profile或者.bashrc添加

export PATH="$PATH:/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/usr/games:/usr/local/games"
source profile 
source .bashrc

给snap设置代理，注意地址不能带https
启动vpn后

[user@lc]_ sudo snap set system proxy.http="http://<proxy_addr>:<proxy_port>" 
[user@lc]_ sudo snap set system proxy.https="http://<proxy_addr>:<proxy_port>"

例如

sudo snap set system proxy.http="http://127.0.0.1:8889"
sudo snap set system proxy.https="http:127.0.0.1:8889"
